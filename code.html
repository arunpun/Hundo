The Code for Hundo


                    //Get values from the page, start function
function getValues(){
    //Get values
    let startValue = document.getElementById("startValue").value;
    let endValue = document.getElementById("endValue").value;

    //Convert string to int
    startValue = parseInt(startValue);
    endValue = parseInt(endValue);

    //Validation
    if(Number.isInteger(startValue) && (Number.isInteger(endValue))){
        //Call generate numbers
        let numbers = generateNumbers(startValue, endValue);
        //Call display numbers
        displayNumbers(numbers);
    } else {
        alert("Error: You must enter integers!");
    }   
}

//Generate numbers from start value to the end value, logic function
function generateNumbers(sValue, eValue){ 
    let numbers = []

    //Get all numbers from start to end
    //For loop - start, end, increment
    for(let i = sValue; i <= eValue; i++){
        numbers.push(i);
    }
    return numbers;
}

//Display the numbers and mark even numbers in bold, view function
function displayNumbers(numbers){
    //To store table data
    let templateRows = "";

    //Loop over the array
    numbers.forEach(element => {
        if(element % 2 == 0){
            templateRows += `<tr><td><strong>${element}</strong></td></tr>`;
        } else {
            templateRows += `<tr><td>${element}</td></tr>`;
        }
    });
    document.getElementById("results").innerHTML = templateRows;
}

The code is structured in three functions, getValues(), generateNumbers() & displayNumbers().

getValues()

This function is triggered by an event listener document.getElementById("btnSubmit").addEventListener("click", getValues when the 'Display the Hundo' button is clicked in the app page. The function creates two variables, startValue and endValue whose values are respectively passed from the input fields using the document.getElementById().value method. These values are then converted to an integer with the parseInt() function. A further type validation is done using a boolean logic in a if-else statement. This statement uses the Number.isInteger() method which checks if the given argument is a number and returns either True or False. Because we have an AND gate, the expression requires both of the statements to evaluate to True which executes the next line of code in the block, generateNumbers(startValue, endValue) and displayNumbers(numbers) otherwise the user is alerted with an appropriate message.

generateNumbers()

This function takes two parameters which I have defined as sValue (start value) & eValue (endvalue). The purpose of this function is to use these parameters and generate all the numbers between them. An empty array called numbers is defined at the start of the function which will store all of the generated numbers. In a for loop, the starting value, i, is set to the value of sValue which is passed as the argument. Similarly, the value of the endValue is used for the condition of the for loop to terminate, i.e. i <= endValue. The i++ increments the value of i until the condition is met. For each loop, the value of i is stored in the numbers array using the array method, numbers.push(i). The function then returns the array after the loop terminates.

displayNumbers()

This function takes the numbers array as an argument and defines an empty string, templateRows which will hold HTML elements, table row and table data. Now that we have an array of numbers, we can use the method, numbers.forEach(element) which loops through each elements in the array, in this case the elements are the generated numbers. In this loop, I have enclosed a if-else statement to check if the number is even using the % operator. This operator returns the remainder hence if the remainder is 0, the number is even as it is cleanly divisible by 2! With their respective HTML tags, the loop stores the table row and table data with each of the numbers in the templateRows as strings, bolding all of the even numbers. This is achieved using template literals, `<tr><td><strong>${element}</strong></td></tr>` where ${element} is dynamically assigned each of the numbers in the array. Because we now have valid HTML stored in templateRows, we can display the numbers using the document.getElementById("results").innerHTML = templateRows; method via DOM manipulation.
